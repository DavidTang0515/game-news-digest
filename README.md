# Game News Digest

每日游戏新闻摘要仓库。内容聚焦游戏行业和玩家社区，不整理泛科技、社会新闻或无关娱乐新闻。

## What Gets Covered

日报只覆盖游戏相关内容：

- 主机、PC、手游、独立游戏新作与更新
- 游戏公司、发行商、平台方、订阅服务与商店政策
- 版号、监管、财报、收购、裁员等行业动向
- 电竞赛事与俱乐部动态
- 游戏改编影视动画、游戏音乐、线下活动等强相关内容
- 国内外玩家社区正在讨论的游戏热点

## Daily Report Format

每天 09:00 自动生成当天日报，文件名格式：

```text
news-YYYY-MM-DD.md
```

每篇日报应包含：

1. 今日最重要新闻
2. 其他值得关注新闻
3. 国内玩家/网友态度
4. 国外玩家/网友态度
5. 影响判断
6. 来源链接
